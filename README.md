## Overview
CeloHT.github.io is the official public website of CeloHT — a global, community‑driven Web3 initiative focused on blockchain education, financial inclusion, agent network development, and environmental reforestation programs.  
The platform is built with Next.js 14, TypeScript, Tailwind CSS, and a modern architecture optimized for performance, accessibility, security, and international‑grade SEO.

## Mission
CeloHT’s mission is to make blockchain technology accessible, useful, and sustainable for communities worldwide through:
- Web3 Education  
- Financial Literacy  
- Digital Payments  
- Local Agent Networks  
- Environmental & Reforestation Programs  

## Core Pillars
### 1. Education
Accessible training on blockchain, cUSD, Valora, digital security, and modern financial tools.

### 2. Agent Network
A structured network of trained community agents supporting users, merchants, and local digital adoption.

### 3. Reforestation
Environmental initiatives including tree‑planting, climate education, and sustainable development projects.

## Technology Stack
| Layer | Tools |
|-------|-------|
| Frontend | Next.js 14 (App Router), TypeScript |
| Styling | Tailwind CSS, PostCSS |
| Content | Markdown, MDX |
| CI/CD | GitHub Actions (CI, Deploy, CodeQL, Lighthouse) |
| Hosting | Vercel |
| Quality | ESLint, Prettier, Husky, Commitlint |
| Testing | Jest, Playwright/Cypress |
| SEO | Sitemap, Robots, Open Graph, Structured Data |

## Repository Structure
- `.github/` — Issue templates, workflows, CODEOWNERS, dependabot  
- `.husky/` — Git hooks (pre‑commit, commit‑msg, pre‑push)  
- `.vscode/` — IDE settings & recommended extensions  
- `public/` — Icons, images, manifest, SEO assets  
- `src/app/` — All website pages (about, education, agents, reforestation, impact, blog, news, faq, contact, privacy, terms)  
- `src/components/` — UI library (layout, sections, cards, forms, charts, navigation, SEO)  
- `src/content/` — Blog, news, documentation  
- `src/lib/`, `src/hooks/`, `src/utils/`, `src/services/` — Logic, helpers, API clients  
- `tests/` — Unit, integration, e2e tests  
- `docs/` — Architecture, development, deployment, security, SEO, performance, accessibility, style guide  
- Root configs — Next.js, Tailwind, TypeScript, PostCSS, Vercel, npm, Prettier, ESLint
